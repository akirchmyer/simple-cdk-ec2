# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.2 - 1.0.4 | :white_check_mark: |
| < 1.0.2  | :x:              |

## Reporting a Vulnerability

Please open an issue if you find a vulnerability and I will fix it right away. Feel free to open a pull request as well.
